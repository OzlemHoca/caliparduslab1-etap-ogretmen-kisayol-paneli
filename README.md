# ÇalıPardusLab1 ETAP Öğretmen Kısayol Paneli

Bu proje, ETAP akıllı tahta ortamında öğretmenin sık kullandığı işlemlere daha hızlı erişmesini sağlamak amacıyla geliştirilmiş yardımcı bir araçtır.

## Proje Amacı

Akıllı tahta kullanılan ders ortamlarında öğretmenler bazı temel işlemleri hızlı ve sade biçimde gerçekleştirmek ister. Bu proje, öğretmenin ders sırasında ihtiyaç duyabileceği temel işlevlere tek noktadan erişmesini sağlayacak bir kısayol paneli mantığı oluşturmak amacıyla hazırlanmıştır.

## Hedefler

- Öğretmenin sık kullandığı işlemlere hızlı erişim sağlamak
- Ders sırasında zaman kaybını azaltmak
- ETAP kullanım deneyimini sadeleştirmek
- Eğitim ortamına uygun modüler bir yardımcı araç geliştirmek

## Temel Özellikler

- Kısayol paneli menüsü
- Sık kullanılan işlemlere hızlı erişim
- Basit terminal tabanlı prototip yapı
- Geliştirilmeye uygun modüler tasarım

## Hedef Kullanıcılar

- Öğretmenler
- Akıllı tahta kullanan eğitim kurumları
- ETAP ve Pardus ortamında çalışan kullanıcılar

## Kullanılan Teknolojiler

- Python 3

## Çalıştırma

Aşağıdaki komut ile çalıştırılabilir:

python3 src/etap_ogretmen_kisayol_paneli.py
